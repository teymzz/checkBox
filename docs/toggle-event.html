CheckBox.js


Toggle Event


Toggle events are events that are expected to run when the checkbox is toggled. Assuming we have a checkbox as shown below:

Custom CheckBox
Copy 
<div checkbox="">
  <div marker="" style="background-color:white"></div> 
  <div marker="" style="background-color:lime"></div>
</div>
<input type="checkbox"> <br> 
<div class="response"></div>

We can setup a global toggle callback function as shown below

Global Toggle Callback Function
Copy 
let checkbox = new CheckBox(true);

checkbox.check({

  toggle: function(checkbox){
   
     let responseField =  document.querySelector('.response');
     
     responseField.innerHTML = 'checked: '+checkbox.checked;
   
  }
  
});

From the sample above, the toggle function will run when the custom checkbox is toggled.

HTML View (Toggle on checkbox)
 

Alternatively, we can set up a local custom callback function using the data-func attribute. A sample format is shown below:

Local Toggle Callback Function
Copy 
<div checkbox="" data-func="ontoggle">
  <div marker=""  style="background-color:white"></div> 
  <div marker=""  style="background-color:lime"></div>
</div>
<input type="checkbox"> <br> 
<div class="response"></div>

From the sample html structure above, the data-func attribute was used to define a local toggle callback function. Now, we can add the defined callback function in a similar format as shown below:

Custom Toggle Function
Copy 
function ontoggle(checkbox){
 
   let responseField =  document.querySelector('.response');
   
   responseField.innerHTML = 'checked: '+checkbox.checked;
 
}
