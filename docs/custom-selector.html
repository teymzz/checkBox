CheckBox.js


Custom Selector


By default, the custom checkbox is identified through the “checkbox” attribute. We can easily change this selector by declaring a custom one. An example of this is shown below:

HTML JS CSS
Copy 
<div checkers="">
  <div marker=""></div> 
  <div marker=""></div>
</div>
<input type="checkbox">

let checkbox = new CheckBox(true);

checkbox.check({

  target: '[checkers]' //custom box selector

});

From the javascript code sample above, the [checkers] selector was used to select the target html checkbox which will overide the default [checkbox] selector. This can be good for localizing checkbox selection and also avoiding conflicts if necessary.
