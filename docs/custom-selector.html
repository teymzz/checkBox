Custom Selector


By default, the custom checkbox is identified through the “checkbox” attribute. We can easily change this selector by declaring a custom one. Assuming we have an html structure like below:

Custom CheckBox
Copy 
<div checkers="">
  <div marker=""></div> 
  <div marker=""></div>
</div>
<input type="checkbox">

We can easily overide the default checkbox custom box selector for the new checkers attribute selector defined above using the format below.

Script
Copy 
let checkbox = new CheckBox(true);

checkbox.check({

  target: '[checkers]' //custom box selector

});

From the sample above, the [checkers] attribute will overide the default selector. This can be good for localizing checkbox selection if necessary.
