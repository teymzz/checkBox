Flipping Markers


Double markers can be easily flipped to switched by setting the flip global configuration option or the data-flip local configuration attribute on the custom checkbox. This feature makes one marker hidden at a time while the other is visible. The value of this option is either true or false. The true value means that only one marker will be displayed at a time. For example, the first marker will be displayed when the checkbox is toggled off while the second marker will be displayed when the checkbox is toggled on. However, the false value means that both markers will be visible at the same time.

Global Configuration Sample
Copy 
<script>

new CheckBox({
    flip: true // toggle double markers
});

</script>

We can equally setup a local configuration using the data-flip attribute as shown below :

Local Configuration
Copy 
<div checkbox="" data-flip="true">
  <div marker=""></div> 
  <div marker=""></div>
</div>
<input type="checkbox"> <br>

This integrated feature makes it easier to create a fast toggle effect for double markers.
