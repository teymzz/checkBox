Flipping Markers


Double markers can be flipped by setting the flip global configuration option or the data-flip local configuration attribute on the custom checkbox. The value of this option is either true or false. The true value means that only one marker will be displayed at a time. For example, the first marker will be displayed when the checkbox is toggled off while the second marker will be displayed when the checkbox is toggled on. However, the false value means that both markers will be visible at the same time.

Global Configuration Sample
Copy 
<script>
new CheckBox({
    flip: true // auto flip markers
});
</script>

We can equally setup a local configuration using the data-flip attribute as shown below :

Local Configuration
Copy
