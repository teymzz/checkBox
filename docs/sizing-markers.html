Sizing Markers


We have already seen how we can style markers using css from earlier samples. We can also achieve a similar sizing effect globally when we initialize the plugin. This global sizing can be applied in different ways as shown below:

JS#1 JS#2 JS#3 
let checkbox = new CheckBox(true);

checkbox.check({

  size: '10px' //set marker width and height

});

let checkbox = new CheckBox(true);

checkbox.check({

  size: {
    width: '10px' //set markers width
    height: '15px' //set markers height
  }
  
});

let checkbox = new CheckBox(true);

checkbox.check({

  size: {
    x: '10px' //set markers width
    y: '15px' //set markers height
  }
  
});

The sample scripts above shows various ways through which the width and height of checkbox markers can be defined by default when initializing the checkboxes. These defaults can be also overridden through the use of the data-size html attribute as shown below :

HTML 
<div checkbox="" data-size="10px|30px">
  <div marker=""></div>
  <div marker=""></div> 
</div>
<input type="checkbox">

In the sample html structure above, the data-size will set the width and height of both of the markers respectively. The left value sets the width, while the right value sets the height. However, if a single value is supplied, it will be used for both the width and the height.
