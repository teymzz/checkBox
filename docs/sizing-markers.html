Sizing Markers


We have already seen how we can style markers using css. We can also achieve a similar sizing effect when we initialize the plugin. This is shown below:

Script
Copy 
let checkbox = new CheckBox(true);

checkbox.check({

  size: '10px' //set marker width and height

});

The string value above will set the size for both width and height of markers, if they exist. However, we may need to specify a different width and height, this can be done as shown below:

Script
Copy 
let checkbox = new CheckBox(true);

checkbox.check({

  size: {
    width: '10px' //set markers width
    height: '15px' //set markers height
  }
  
});

The sample above will set both the width and height of all selected markers on the web page. We can do this similarly by using an x and y axis to specify the width and height of the markers respectively.

Script
Copy 
let checkbox = new CheckBox(true);

checkbox.check({

  size: {
    x: '10px' //set markers width
    y: '15px' //set markers height
  }
  
});

Any of the structure sample above is accepted by the checkbox plugin. This configuration will affect only relatively selected markers on the webpage. We can however apply a much better approach using the html attributes supported for setting sizes just as shown below:

CheckBox (Sizing)
Copy 
<div checkbox="" data-size="10px|30px">
  <div marker=""></div>
  <div marker=""></div> 
</div>
<input type="checkbox">

In the sample html structure above, the data-size will set the width and height of both of the markers respectively. The left value sets the width, while the right value sets the height. However if a single value is supplied, it will be used for both the width and the height.
