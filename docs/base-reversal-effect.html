Base Effects


The polar toggle effect is a bit similar to radio button effect except that this relatioship is only between the two markers of a single custom checkbox. In this situation, when a marker is active, it cannot toggle itself off because it will require the action of the second marker to switch. In order to achieve this effect, both markers will be visible. Hence, data-flip needs to be set as false and also, the data-control attribute will be set as ":markers" while the markers will be manually designed using css.

checkbox (Base Effect)
In the sample above, the first checkbox is the parent checkbox while the second and third checkboxes are children of the parent checkbox, hence, they must both be active before the parent checkbox is active. Whenever the parent checkbox is toggled on or off, the other two children checkboxes will respond similarly. If both of the children checkboxes are not toggled on, then the parent checkbox is never toggled on. However, the last checkbox is the inverse of the parent checkbox and the second and third checkboxes. This means that when the status of the parent checkbox and the two children checkboxes are the same, the last checkbox will be the inverse of that.
