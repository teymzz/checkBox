Init Event


Init event is an in-built callback event that is expected to run immediately the plugin is initialized or instantiated. This event occurs immediately the page loads before any user activity is detected on the checkbox. Assuming we have a custom checkbox in the format below:

HTML 
<div checkbox="">
  <div marker=""></div> 
  <div marker=""></div>
</div>
<input type="checkbox"> <br> 
<div class="response"></div>

We can setup an instance function as shown below

JS 
let checkbox = new CheckBox(true);

checkbox.check({

  init: function(checkbox){
   
     let responseField = document.querySelector('.response');
     
     responseField.innerHTML = 'CheckBox Plugin Initialized';
   
  }
  
});

From the sample above, the init function will run when the page loads.
