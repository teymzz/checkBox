CheckBox.js


Toggle Attributes : checked


We can specify that a custom checkbox should assign the value from an attribute into another attribute when the custom checkbox is checked and should remove it when it is unchecked. In order to do this, let's create a sample html structure below.

Sample Html Structure
Copy 
<div checkers class="foo" data-switch="bar">
<div marker style="background-color: #594fc8;"></div>
</div>
<input type="checkbox">

From the sample html structure, assuming we need to add the bar value into the class attribute when a checkbox is checked, we can easily achieve this using the sample script below :

Global: assign value on checked
Copy 
<script>
new CheckBox({
assign: {
    target: '[checkers]',
    attr: ['data-switch', 'class', 'checked']
}    
});
</script>

Alternatively, we can do this locally using the data-assign attribute

Local Assign Sample Format 
<div checkers class="foo" data-switch="bar" data-assign="checked|data-switch|class">
<div marker style="background-color: #594fc8;"></div>
</div>
<input type="checkbox">

From the sample above, the bar value will be assigned into the class attribute when the checkbox is checked and it will be removed when the checkbox is unchecked. To see the effect of this, let's add a custom css below

Custom css
Copy 
[checkers] {
  width: fit-content;
  border-radius: 5px;    
  border: solid; 
  padding: 4px;    
}

[checkers] > [marker] {
width: 20px;
height: 20px;    
background-color: transparent;
transition: width .5s ease-in-out, 
            background-color .5s ease-in-out;
}

[checkers] > [marker] {
width: 50px;
background-color: green;
}

.bar[checkers] > [marker] {
width: 30px;
background-color: red;
}

The sample css above was added to give a little width and background color animation effect when the value is toggled in and out of the custom checkbox. The html response is shown below:
