Rating effect


The rating toggle effect includes a list of custom checkboxes that exists together purposely for rating items. In this effect, when a checkbox is checked, all relative custom checkboxes to its left side are automatically toggled on while those to the right are toggled off. This is achieved by setting data-bind attribute to "rating". An example of this is shown below:

Rating Effect HTML CSS JS

Naturally, it is difficult to detect the number of checkbox that are checked due to the structure of the plugin. However, this data is made obtainable by the plugin through the checkedBoxes method which provides the exact number of checkboxes that are currently checked.
