Markers (single)


The sample below best reveal the format in which a custom checkbox can be designed with a single marker.

HTML JS CSS 
<div checkbox>
  <div marker=""></div>
</div>
<input type="checkbox">

[checkbox] {
    width: fit-content;
    border-radius: 5px;
    border: solid; 
    padding: 4px;
}

[checkbox] > [marker] {
    width: 20px;
    height: 20px;
}

[checkbox][checked="unchecked"] {
    border-color: white;
}

[checkbox][checked="checked"] {
    border-color: lime;
}

[checkbox][checked="unchecked"] > [marker] {
    background-color: white;
}

[checkbox][checked="checked"] > [marker] {
    background-color: lime;
}

<script>

new CheckBox(); //initialize and execute

</script>

From the sample above, we can see how to apply a single marker on the custom checkbox. Whenever the custom checkbox is checked, an attribute checked is automatically assigned to it based on the current status of its native checkbox. This attribute assumes a value that is either "checked" or "unchecked". This makes it easier to style the custom checkbox using the relative target selector as in the case of the css above.
