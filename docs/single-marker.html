CheckBox.js


Markers (single)


Previously, we learnt that the way to structure a single-marker custom checkbox is similar to the structure below:

CheckBox (Single marker)
Copy 
<div checkbox>
  <div marker=""></div>
</div>
<input type="checkbox">

Let's add some styling to the custom checkbox and marker.

Custom css
Copy 
[checkbox] {
    width: fit-content;
    border-radius: 5px;
    border: solid; 
    padding: 4px;
}

[checkbox] > [marker] {
    width: 20px;
    height: 20px;
}

[checkbox][checked="unchecked"] {
    border-color: white;
}

[checkbox][checked="checked"] {
    border-color: lime;
}

[checkbox][checked="unchecked"] > [marker] {
    background-color: white;
}

[checkbox][checked="checked"] > [marker] {
    background-color: lime;
}

HTML View (Toggle on checkbox)

From the above, we can see how to apply a single marker on the custom checkbox. The value of the attribute checked is automatically assigned to the custom checkbox based on the current status of its native checkbox. This makes it easier to style the custom checkbox.
