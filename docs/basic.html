CheckBox.js


Basic sample


The code sample below is the easiest way to create a custom checkbox.

HTML sample
Copy 
<div checkbox=""></div>

<input type="checkbox">

Custom css
Copy 
[checkbox] {
    width: 20px;
    height: 20px;
}

[checkbox][checked="unchecked"] {
    background-color: white;
}

[checkbox][checked="checked"] {
    background-color: lime;
}

Script
Copy 
<script>
new CheckBox(); //initialize and execute
</script>

HTML View (Toggle on checkbox)

The sample above is the most basic way to handle a custom checkbox without any complex configuration. When we initialize the plugin, all native checkboxes becomes hidden, leaving only their relative custom checkbox.
