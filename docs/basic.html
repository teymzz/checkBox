CheckBox.js


Basic checkbox


The code sample below is the easiest way to create a custom checkbox without any custom configuration.

HTML CSS 
<div checkbox=""></div>

<input type="checkbox">

[checkbox] {
    width: 20px;
    height: 20px;
}

[checkbox][checked="unchecked"] {
    background-color: white;
}

[checkbox][checked="checked"] {
    background-color: lime;
}

Now, we can initialize the checkbox plugin as shown below

JS 
<script>

new CheckBox(); //initialize and execute

</script>    



 
 
//toggle on checkbox
HTML View (Toggle on checkbox)

When we initialize the plugin as shown above, all native checkboxes becomes hidden, leaving only their relative custom checkbox that immediately precedes them.
