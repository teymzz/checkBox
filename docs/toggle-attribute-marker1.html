CheckBox.js


Toggle Attributes - :marker1


When using markers, can specify that a custom checkbox should assign the value from an attribute into another attribute on the the first custom checkbox marker when the checkbox is checked. In order to do this, let's create a sample html structure below.

Sample Html Structure
Copy 
<div checkers class="foo" data-switch="bar">
<div marker style="background-color: red;"></div>
<div marker style="background-color: green;"></div>
</div>
<input type="checkbox">

From the sample html structure, assuming we need to add the bar value into the class attribute of the first marker when the checkbox is checked, we can easily achieve this using the sample script below :

Global Assign - :marker1
Copy 
<script>
new CheckBox({
assign: {
    target: '[checkers]',
    attr: ['data-switch', 'class', ':marker1']
}    
});
</script>

Alternatively, we can do this locally using the data-assign attribute

Local Assign - :marker 1 
<div checkers class="foo" data-switch="bar" data-assign=":marker1|data-switch|class">
<div marker style="background-color: red;"></div>
<div marker style="background-color: green;"></div>
</div>
<input type="checkbox">

From the sample above, the bar value will be added into the class attribute of the first marker when the checkbox is checked and it will be removed when the checkbox is unchecked. To see the effect of this, let's add a custom css below

Custom css
Copy 
[checkers] {
    width: fit-content;
    border-radius: 100vh;    
    border: dotted 2.5px #514e68; 
    padding: 4px;    
}

[checkers] > [marker] {
    width: 60px;
    height: 2px;    
    background-color: transparent;
    transition: width .5s ease-in-out, 
                background-color .5s ease-in-out;
}

[checkers] > [marker]:first-child {
    width: 30px;
    background-color: rgba(149, 15, 15, 0.66);
}

[checkers] > .bar[marker] {
    width: 120px;                            
    background-color: red;
}

[checkers] > [marker] {
    border-radius: 100vh;
    background-color: green;
    margin: 4px;
}

The sample css above was added to give a little width and background color animation effect when the value is toggled in and out of the custom checkbox. The html response is shown below:

HTML View (Toggle on checkbox)
