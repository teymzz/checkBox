Markers (double)


The double markers is similar to the single markers except that two markers is added into the custom checkbox container instead of a single marker.

HTML CSS 
<div checkbox>
  <div marker=""></div> <!-- @unchecked -->
  <div marker=""></div> <!-- @checked -->
</div>
<input type="checkbox">

[checkbox] {
    width: fit-content;
    border-radius: 5px;
    border: solid;
    padding: 4px;
}

[checkbox] > [marker] {
    width: 20px;
    height: 20px;
}

[checkbox][checked="checked"] > [marker]:nth-child(1) {
    opacity: .1;
}

[checkbox]:not([checked="checked"]) > [marker]:nth-child(2) {
    opacity: .1;
}

[checkbox] > [marker]:nth-child(1)  {
    background-color: white;
}

[checkbox] > [marker]:nth-child(2) {
    background-color: lime;
}

From the structure above, the plugin automatically detects that we have double markers and creates a toggle effect for each marker. The first marker will be displayed when the checkbox is unchecked while the second marker will be displayed when the checkbox is checked.
