Binding Parent


A custom checkbox can be binded to its top level closest parent thereby, making the parent the checkbox controller. In order to do this, the custom checkbox must specify its parent controller using the data-control attribute and setting the value to a valid parent selector.

HTML CSS JS 
<button parent>
  <div data-role="checkbox">
      <div checkbox data-control="[parent]" data-size="20px" data-color="@red|green">
          <div marker>
              <div class="fit b-currentColor"></div>
          </div>         
      </div>
      <input type="checkbox">
  </div>
</button> 

button {
  padding: 10px;
  border-radius: 100vh;  
  border: none;  
  display: inline-flex;  
  justify-content: center;  
  align-items: center;    
  gap: .5em;    
}

button > * {
  display:inline-flex;
}

button > *:first-child {
  border: solid 2px #3a3446;
}

button [checkbox] {
  border: solid 4px transparent;
  display: inline-flex;
}

button :where(:first-child,[marker],.fit) {
  border-radius: 100vh;
}

button .text-field {
  display: flex;
  justify-content: center;
  align-items: center;
}

.b-currentColor {
  background-color: currentColor;
}

new CheckBox();


In the sample above, when the parent button is clicked, the toggle event will be triggered for the custom checkbox.
