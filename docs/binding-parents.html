CheckBox.js


Binding Parent


A custom checkbox can be binded to its top level closest parent thereby, making the parent the checkbox controller. In order to do this, the custom checkbox must specify its parent controller using the data-control attribute and setting the value to a valid parent selector.

HTML CSS JS 
<button parent class="in-flex mid" style="padding: 10px; gap:.5em">
    <div data-role="checkbox" class="in-flex wrap">
        <div checkbox data-control="[parent]" data-size="20px" data-color="@red|green" class="in-flex">
            <div marker>
                <div class="fit b-currentColor"></div>
            </div>         
        </div>
        <input type="checkbox" value="W">
    </div>
</button> 

[data-role="checkbox-list"] {
  display: inline-flex;
  gap: .5em;  
}
  
[data-role="checkbox-list"] [marker] {
  width: 20px;
  height: 20px;  
}
  
[checkbox] {
  display: inline-flex;
  border: dotted 2px;  
  border-radius: 100vh;  
  padding: 4px;  
}
  
[checkbox] [marker] {
  width: 20px;
  height: 20px; 
  border-radius: inherit;    
  background-color: currentColor;    
}

.rotate[checkbox][checked="checked"] {
    animation: rotate 5s infinite linear;;
}

@keyframes rotate {

    0% {
      transform: rotate(0deg);
    }
    
    100% {
      transform: rotate(360deg);
    }
    
}
Checkbox: (data-control)


In the sample above, when the parent button is clicked, the toggle event will be triggered for the custom checkbox.
