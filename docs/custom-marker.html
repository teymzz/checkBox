CheckBox.js


Custom Marker


By default, the custom checkbox marker is selected through the “[marker]” attribute selector. We can easily overide this selector by declaring a new custom selector. Assuming we have an html structure like below:

HTML Custom Markers
Copy 
<div checkbox="">
  <div btn=""></div> 
  <div btn=""></div>
</div>
<input type="checkbox">

We can easily overide the default [marker] selector for a new [btn] attribute selector using the format below :

JS Custom Marker Selection
Copy 
let checkbox = new CheckBox(true);

checkbox.check({

  marker: '[btn]' //custom marker selector

});

From the sample above, the [btn] attribute will overide the default marker selector.
