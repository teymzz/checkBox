CheckBox.js


Adding Markers


A marker is a supported control button that lies within any custom checkbox element that is identified with the "marker" attribute. The checkbox plugin provide supports for either one or a maximum of two markers. We can define a single marker as shown below:

CheckBox (Single marker)
Copy 
<div checkbox>
  <div marker=""></div> <!-- marker -->
</div>
<input type="checkbox">

Similarly, we can also define a double marker using the structure below:

CheckBox (Double markers)
Copy 
<div checkbox>
  <div marker=""></div>
  <div marker=""></div>
</div>
<input type="checkbox">

Internally, the checkbox plugin detects the number of control markers supplied for any custom checkbox. This means that single markers are handled differently from double markers. Let's take a look at how single markers are handled on the next page.
