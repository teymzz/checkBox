CheckBox.js


Markers (double)


The double markers is similar to the single markers except that two markers are provided which automatically creates a toggle effect. The sample structure is shown below

CheckBox (Double markers)
Copy 
<div checkbox>
  <div marker=""></div> <!-- @unchecked -->
  <div marker=""></div> <!-- @checked -->
</div>
<input type="checkbox">

From the structure above, the plugin automatically detects that we have double markers and creates a toggle effect for each marker. The first marker will be displayed when the checkbox is unchecked while the second marker will be displayed when the checkbox is checked. Using css, we can also set the sizes for each marker or redesign the custom checkbox itself.

Custom css
Copy 
[checkbox] {
    width: fit-content;
    border-radius: 5px;
    border: solid;
    padding: 4px;
}

[checkbox] > [marker] {
    width: 20px;
    height: 20px;
}

[checkbox] > [marker]:nth-child(1)  {
    background-color: white;
}

[checkbox] > [marker]:nth-child(2) {
    background-color: lime;
}
