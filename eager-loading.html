Eager Loading


Eager loading is necessary in situations where we need to preload a toggle event. We have already seen how to initialize an event using the init option or the data-init attribute. Also, we have learnt about the toggle option or data-func attribute. We can easily combine these features to create a smart eager loading for the checkbox toggling. Assuming we have an html script below:

We can setup an eager-loading toggle function as shown below :

sample: toggle
Copy 
<script>
new CheckBox({
    init: function(checkbox) {
        
        let value = checkbox.value;

        if( checkbox.checked ) {
            console.log(`checkbox ${value} is: checked`)
        } else {
            console.log(`checkbox ${value} is: unchecked`)
        }

    },
    toggle: function(checkbox) {
        
        checkbox.init(checkbox);

    }
});
</script>

From the above, we can set that the argument supplied into the toggle() callback option has access to the init() callback option. This makes it easier to run the init() within the toggle() function rather than redefining it. This means that init() function will first run when the page loads and it will continue to run when the checkbox is toggled.
